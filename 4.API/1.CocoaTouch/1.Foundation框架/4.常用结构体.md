
### 常用结构体

#### NSRange结构体
NSRange的定义如下：
```obj
    typedef struct _NSRange {
        NSUInteger location;
        NSUInteger length;
    } NSRange;
```
NSRange表示一个范围，第一参数是起始位置，第二个参数是长度。

```obj
    void NSRangeTest(void){
        NSRange range = {3,5};      // 声明和初始化
        range.location = 4;         // 修改
        range.length = 8;
        // NSStringFromRange 转化成字符串
        NSLog(@"range is %@",NSStringFromRange(range));
    }
```
